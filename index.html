5. 네트워크 검증 가이드

  1. DevTools → Network 탭 열기
  2. Preserve log ✓
  3. Filter: sync_js
  4. 페이지 전체 새로고침 → 10초 관찰
  5. 통과 기준: 분석 SDK 쪽 요청만 있고, 운영 SDK(`TDApp`) 쪽에서 추가로 생성되는 /sync_js 요청이 없음. autoTrack: {all false} 설정 덕분.

6. GTM 템플릿 설치 방법 (고객용)

  1. GTM 컨테이너 → Templates → Tag Templates → New
  2. 우상단 ⋮ → Import → TE Operation_KR.tpl 또는 EN/JP/CN 언어 파일 선택
  3. Save
  4. Tags → New → "AE Operation" 선택 → 파라미터 설정 → Save
